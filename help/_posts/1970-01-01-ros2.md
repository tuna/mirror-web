## ROS2 镜像使用帮助

输入如下命令，下载 ROS 的 GPG Key：

```bash
sudo apt install curl gnupg2
sudo curl -sSL https://{{ site.hostname }}/rosdistro/ros.key  -o /usr/share/keyrings/ros-archive-keyring.gpg
```


再输入如下命令，添加软件仓库，并更新索引：
